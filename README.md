# Flutter Basics

A new Flutter project for the sample course.

## About the project

This projects shows the usage of some Flutter Widgets to develop Flutter apps.

Some of the used Widgets in the project are:

- Scaffold
- Buttons
- Text
- Container
- Column / Row
- ListView
- Image
